# 📚 雪梨拼读 - 背单词应用

基于艾宾浩斯记忆曲线的英语拼写练习应用。

## 功能特点

- 📝 手动输入拼写，拒绝选择题
- 🧠 艾宾浩斯记忆曲线，智能复习
- 💡 答错时提供提示
- 📱 支持 iPad/iPhone，随时随地学习

## 访问地址

打开浏览器访问：https://innerpeace0522.github.io/spelling-app

## 使用说明

1. 选择想要练习的章节（Level 1/2/3）
2. 根据中文释义输入正确的英文拼写
3. 答对会显示绿色，答错会给出提示
4. 系统会根据记忆曲线自动安排复习

## 数据存储

所有学习进度保存在本地浏览器中（localStorage）。
